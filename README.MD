# Jquery.Letterize

Used to split a textnode into an array of spans.

## Install

```sh
bower install jquery.letterize
```

## Usage

```javascript
$('<div>Letters</div>').letterize();
// <div>
//   <span>L</span>
//   <span>E</span>
//   <span>T</span>
//   <span>T</span>
//   <span>E</span>
//   <span>R</span>
//   <span>S</span>
// </div>
```
